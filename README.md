FreeBSD [HAMMER2](https://gitweb.dragonflybsd.org/dragonfly.git/blob/HEAD:/sys/vfs/hammer2/DESIGN)
========

## Requirements

+ FreeBSD 14.x

+ src tree under /usr/src

## Build

        $ cd freebsd_hammer2
        $ make

## Install

        $ cd freebsd_hammer2
        $ make install

## Uninstall

        $ cd freebsd_hammer2
        $ make uninstall
